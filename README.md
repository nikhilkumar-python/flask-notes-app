# flask-notes-app
# Flask Notes Application

This is a simple Flask-based notes application.

## Features
- Add notes
- View all notes
- Edit notes
- Delete notes
- SQLite database
- Full CRUD operations

## Technologies Used
- Python
- Flask
- HTML
- SQLite

## How to Run
1. Install Flask
2. Run `python app.py`
3. Open browser and go to `http://127.0.0.1:5000/`

## Purpose
This project was built to understand Flask routing, form handling,
database operations, and CRUD concepts.
